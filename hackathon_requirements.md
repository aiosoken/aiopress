# Google Cloud Japan AI Hackathon Vol.4 調査結果

## ハッカソン概要

- **名称**: 第4回 Agentic AI Hackathon with Google Cloud
- **テーマ**: 進化するAIと共に。君だけの「エージェント」を創り出そう。
- **期間**: 2025年12月10日～2026年2月15日
- **賞金総額**: 175万円
- **主催**: Zenn（クラスメソッド株式会社）
- **協賛**: グーグル・クラウド・ジャパン合同会社

## 重要なポイント

### 1. テーマ: Agentic AI
- **自律的に考え行動するAI**（エージェント）の開発がテーマ
- 最新鋭モデル「Gemini 3」を活用できる
- Agent Development Kit (ADK) の活用が推奨されている

### 2. 技術要件
- **Google Cloudの使用が必須**
- Google Cloudクレジット300ドル分のクーポンが提供される
- Firebase、Vertex AI、Gemini APIなどの活用が想定される

### 3. 審査基準
- 「課題発見」「アイデア」「実装」のどこかが光っていれば受賞可能
- **奨励賞（5枠）** が新設され、評価のチャンスが拡大

### 4. スケジュール
| 項目 | 期間 |
|:---|:---|
| 登録および提出期間 | 2025年12月10日～2026年2月15日 |
| 1次審査期間 | 2026年2月16日～2026年2月23日 |

### 5. サポートイベント
- **AI Bootcamp**: Firebase、ADK、Vertex AIなどのハンズオン（1月19日～）
- **オンラインオフィスアワー**: 技術的な質問対応（1月8日）
- **チームビルディングイベント**: 1月16日

## AIOプレスの適合性分析

### ✅ 適合するポイント
1. **Google Cloud技術の活用**: Firebase + Vertex AI + Gemini APIを使用
2. **Agentic AI要素**: ブランド資産を自律的に分析し、最適なクリエイティブを生成するAIエージェント
3. **実用的な課題解決**: 企業のブランド一貫性とAI時代の可視性という明確な課題に対応
4. **技術スタック**: Next.js + TypeScript + Firebase（ハンズオンで学べる内容と一致）

### 🔧 最適化が必要なポイント
1. **Epson連携**: ハッカソンではプリンター連携は必須ではない → **オプショナル化**が適切
2. **エージェント性の強調**: 「自律的に考え行動する」という側面をより明確にする必要がある
3. **ADKの検討**: Agent Development Kitの活用を検討すると評価が高まる可能性

## 推奨アプローチ

### コア機能（ハッカソン版）
1. **ブランド資産の自動分析**: Vision AI + Gemini APIによる資産の自動分類・タグ付け
2. **AIエージェントによるクリエイティブ生成**: ブランドガイドラインを学習し、自律的に最適なコンテンツを生成
3. **ブランド一貫性の自動評価**: 生成物がブランドガイドラインに適合しているかをAIが自動判定

### オプショナル機能
- **Epson Connect API連携**: プリンター・スキャナー機能（デモ時のみ使用）

### 技術構成
- **フロントエンド**: Next.js 15 + TypeScript + React 19
- **バックエンド**: Firebase (Authentication, Firestore, Storage, Functions)
- **AI**: Vertex AI + Gemini API + Imagen + Vision AI
- **オプション**: Epson Connect API（プリンター連携）
