## Usage

CSS変数を使用してスタイルを適用できます:

```css
.my-element {
  background-color: var(--primary);
  color: var(--primary-foreground);
}
```

Tailwind CSSでは以下のように使用します:

```jsx
<div className="bg-primary text-primary-foreground">
  Content
</div>
```
