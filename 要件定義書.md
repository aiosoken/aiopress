## 2. プロジェクト概要

### 2.1. アプリケーション名
**AIOプレス** (AI-Optimized Press)

### 2.2. アプリケーションの目的
AIOプレスは、企業のブランド資産を学習し、自律的に最適なブランドコミュニケーションを生成する**AIエージェント**である。Google Cloudの最先端AI技術（Vertex AI, Gemini）を駆使し、企業のブランド一貫性を担保しながら、AI時代におけるマーケティング活動を自動化・最適化することを目的とする。

### 2.3. ハッカソンテーマへの適合性
本ハッカソンのテーマ「**君だけの『エージェント』を創り出そう**」に完全に合致する。AIOプレスは、単なるツールではなく、ブランドのペルソナを理解し、戦略を立て、コンテンツを生成するという一連のタスクを自律的に実行する「ブランド戦略AIエージェント」として機能する。

### 2.4. 対象ユーザー
- **プライマリユーザー**: マーケティング担当者、ブランドマネージャー、クリエイティブディレクター
- **セカンダリユーザー**: 中小企業の経営者、スタートアップの創業者、広告代理店の担当者

## 3. 解決する課題

### 3.1. 課題の整理

| 課題カテゴリ | 具体的な課題 | ビジネスインパクト | 優先度 |
|:---|:---|:---|:---|
| **資産のサイロ化** | 過去に作成された有益な資料（デザイン、コピー、戦略書等）が整理されず、ナレッジとして活用されていない。 | 毎回ゼロから制作するため、デザインコストと工数が膨らみ続ける。 | 高 |
| **ブランドの不統一** | 営業、広告、店舗など、部署ごとに異なるメッセージやデザインが乱立し、ブランドの一貫性が欠如している。 | 収益機会が最大23%失われる可能性がある。[1] | 高 |
| **AI時代の可視性** | AIによる情報収集やコンテンツ生成が主流となる中で、AIに発見・評価されないブランドは顧客の選択肢から除外される。 | AI・検索経由の商品発見率は70%以上に達しており、この潮流から取り残されるリスクがある。[2] | 中 |

### 3.2. 課題解決のアプローチ

1. **資産の一元管理**: Firebase StorageとFirestoreを活用し、ブランド資産を一元管理
2. **AIによる自動分析**: Vision AIとGemini APIにより、資産からブランド特性を自動抽出
3. **一貫性の担保**: 学習したブランド特性に基づき、自動的にブランド一貫性のあるコンテンツを生成
4. **AI最適化**: AI検索で発見されやすい構造化されたコンテンツを生成

## 4. 機能要件

### 4.1. 認証・ユーザー管理機能

#### FR-001: ユーザー認証
- **要件ID**: FR-001
- **優先度**: 必須
- **説明**: ユーザーはEmail/PasswordまたはGoogle OAuthでログインできる
- **詳細**:
  - Firebase Authenticationを使用
  - Email/Password認証をサポート
  - Google OAuth認証をサポート
  - セッション管理機能
- **受入基準**:
  - 新規ユーザーがアカウントを作成できる
  - 既存ユーザーがログインできる
  - ログアウト機能が動作する
  - セッションが適切に管理される

#### FR-002: ユーザープロファイル管理
- **要件ID**: FR-002
- **優先度**: 必須
- **説明**: ユーザーは自身のプロファイル情報を管理できる
- **詳細**:
  - 表示名、メールアドレス、プロフィール画像の設定
  - プロファイル情報の更新機能
- **受入基準**:
  - プロファイル情報の表示・編集が可能
  - 変更内容がFirestoreに保存される

### 4.2. ブランド管理機能

#### FR-003: ブランド作成
- **要件ID**: FR-003
- **優先度**: 必須
- **説明**: ユーザーは新しいブランドを作成できる
- **詳細**:
  - ブランド名、説明、ロゴ画像の設定
  - ブランド作成者は自動的にOWNERロールを付与される
- **受入基準**:
  - ブランドが正常に作成される
  - 作成者がOWNERとして登録される

#### FR-004: ブランドメンバー管理
- **要件ID**: FR-004
- **優先度**: 必須
- **説明**: ブランドOWNERはメンバーを追加・削除・ロール変更できる
- **詳細**:
  - ロール: OWNER, ADMIN, MEMBER
  - メンバー招待機能（メールアドレスで招待）
  - メンバー一覧の表示
- **受入基準**:
  - メンバーの追加・削除が可能
  - ロール変更が反映される
  - メンバー一覧が正しく表示される

### 4.3. ブランド資産管理機能

#### FR-005: 資産アップロード
- **要件ID**: FR-005
- **優先度**: 必須
- **説明**: ユーザーはブランド資産（PDF, 画像, テキスト）をアップロードできる
- **詳細**:
  - 対応形式: PDF, PNG, JPG, JPEG, GIF, TXT, DOCX
  - 最大ファイルサイズ: 10MB
  - ドラッグ&ドロップ対応
  - 複数ファイル一括アップロード対応
- **受入基準**:
  - ファイルが正常にアップロードされる
  - Firebase Storageに保存される
  - アップロード進捗が表示される

#### FR-006: 資産の自動分析
- **要件ID**: FR-006
- **優先度**: 必須
- **説明**: アップロードされた資産は自動的にAI分析される
- **詳細**:

  - Gemini APIによる詳細分析（キーワード、トーン、説明、エンティティ抽出）
  - 分析結果はFirestoreに保存される
- **受入基準**:
  - アップロード後に自動的に分析が開始される
  - 分析結果がFirestoreに保存される
  - 分析結果がUIに表示される

#### FR-007: 資産一覧表示

- **優先度**: 必須
- **説明**: ユーザーはブランドの資産一覧を閲覧できる
- **詳細**:

  - ソート機能（日付順、名前順）

- **受入基準**:
  - 資産一覧が正しく表示される
  - フィルタリング・ソートが動作する
  - 検索が機能する

#### FR-008: 資産削除
- **要件ID**: FR-008
- **優先度**: 必須
- **説明**: ユーザーは資産を削除できる

## 7. ユーザーストーリー

### 7.1. ストーリー1: ブランド資産の一元管理

**As a** マーケティング担当者  
**I want to** 過去のブランド資料を一元管理したい  
**So that** 新しいコンテンツ制作時に過去の資料を参照できる

**受け入れ基準**:
- 複数のファイル形式（PDF, 画像, テキスト）をアップロードできる
- アップロードしたファイルが一覧で確認できる
- ファイルを検索・フィルタリングできる

### 7.2. ストーリー2: ブランド特性の自動抽出

**As a** ブランドマネージャー  
**I want to** アップロードした資産からブランド特性を自動抽出したい  
**So that** 手動でブランドガイドラインを作成する手間を省ける

**受け入れ基準**:
- 資産アップロード後に自動的に分析が開始される
- カラー、トーン、キーワードが抽出される
- 抽出結果がデザインシステムとして保存される

### 7.3. ストーリー3: ブランド一貫性のあるコンテンツ生成

**As a** クリエイティブディレクター  
**I want to** ブランド特性に基づいたコンテンツを自動生成したい  
**So that** ブランド一貫性を保ちながら効率的にコンテンツを制作できる

**受け入れ基準**:
- 自然言語で指示を出せる
- 生成されたコンテンツがブランド特性に適合している
- テキストと画像の両方を生成できる

### 7.4. ストーリー4: チームでのブランド管理

**As a** ブランドOWNER  
**I want to** チームメンバーを追加してブランドを共同管理したい  
**So that** 複数人でブランド資産を管理できる

**受け入れ基準**:
- メンバーを招待できる
- ロール（OWNER, ADMIN, MEMBER）を設定できる
- メンバー一覧を確認できる

| **AIO** | AI-Optimized（AI最適化）の略。AI検索で発見されやすいコンテンツを指す |
| **デザインシステム** | ブランドの視覚的・言語的特性を定義したシステム（カラー、タイポグラフィ、トーン等） |
| **ブランド資産** | ブランドに関連する資料（ロゴ、パンフレット、コピー、戦略書等） |
| **クリエイティブ** | AIが生成したコンテンツ（テキスト、画像等） |
| **AIエージェント** | 自律的にタスクを実行するAIシステム |
| **Vertex AI** | Google CloudのAI/MLプラットフォーム |
| **Gemini API** | GoogleのマルチモーダルAI API |
| **Imagen API** | Googleの画像生成API |
| **Vision AI** | Google Cloudの画像認識・OCR API |
| **Epson Connect API** | Epsonハードウェアと連携するためのクラウドAPI |
